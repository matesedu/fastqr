# Security Policy

## Supported Versions

`fastqr` is pre-1.0. Security fixes target the `main` branch until release branches exist.

## Reporting a Vulnerability

Please report vulnerabilities privately to the maintainers before opening a public issue. Include the affected surface, a minimal reproduction, expected impact, and any relevant environment details.

Do not include secrets in public QR samples. QR codes can encode credentials, Wi-Fi passwords, access tokens, and personal data.
